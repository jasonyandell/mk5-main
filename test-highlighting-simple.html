Manual Highlighting Test

Steps to test:

  1. Open developer console (F12)
  2. Click the link below to open game in bidding phase
  3. Click "Actions" tab at the bottom
  4. Hover over any domino in "Your Hand"
  5. Check console for debug messages
  6. Check if dominoes change color
Open Game (Bidding Phase)

Expected console output when hovering:

    ActionPanel handleDominoHover: { domino: "5-3", isEntering: true, enableSuitHighlighting: true, gamePhase: "bidding" }
    ActionPanel shouldHighlight: { domino: "5-3", hoveredSuit: 5, result: "primary" }
    ActionPanel shouldHighlight: { domino: "5-2", hoveredSuit: 5, result: "primary" }
    ... etc for each domino

Expected visual changes:

  • A badge appears showing "Fives" (or whatever suit you hover)
  • All dominoes with that suit should highlight in color
  • When hovering a double, all doubles highlight plus dominoes with that number
