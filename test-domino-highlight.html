Texas 42 - Domino Highlighting Test

Test Overview

This test verifies that dominoes are highlighted correctly during:

  • Bidding Phase: Dominoes should highlight with suit-specific colors when hovering
  • Trump Selection: Dominoes should highlight generically (yellow/gold) when hovering over trump buttons

✓ Bidding Phase - Suit Color Highlighting

When hovering over dominoes during bidding:

  • Blanks (0) - Gray gradient (#6b7280)
  • Ones (1) - Red gradient (#ef4444)
  • Twos (2) - Orange gradient (#f97316)
  • Threes (3) - Yellow gradient (#f59e0b)
  • Fours (4) - Green gradient (#84cc16)
  • Fives (5) - Blue gradient (#3b82f6)
  • Sixes (6) - Purple gradient (#a855f7)
  • Doubles - Pink/Rose gradient (#ec4899)

The suit indicator badge matches the domino highlighting color.

✓ Trump Selection - Generic Highlighting

When hovering over trump selection buttons:

  • Affected dominoes highlight with generic yellow/gold color
  • Trump button shows hovering state with pointing finger animation
  • No suit-specific coloring is applied

✓ Animation Behavior

Dominoes should NOT re-animate when hovering between them. The initial slide-in animation only plays once when the view loads.

Manual Test Steps

  1. Start the app and navigate to the Actions panel during bidding
  2. Hover over different dominoes and verify suit-specific colors
  3. Make a bid (e.g., 30) to reach trump selection
  4. Hover over trump buttons and verify generic highlighting
  5. Verify no re-animation occurs when moving between dominoes

Implementation Details
