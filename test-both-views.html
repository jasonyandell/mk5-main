Test Domino Highlighting in Both Views

This page tests highlighting in both the Play view and Actions view during bidding phase.

Instructions:

  1. Click the link below to open a game in bidding phase
  2. Try hovering over dominoes in the "Play" tab (bottom nav)
  3. Then switch to "Actions" tab and hover over dominoes there
  4. Check if dominoes change color when you hover

Test Link:

Open Game in Bidding Phase

Expected Behavior:

  • When hovering over a domino, related dominoes should highlight in yellow/orange
  • A badge should appear showing which suit is being highlighted
  • For doubles: all doubles highlight as primary, and dominoes with that number highlight as secondary
  • For non-doubles: all dominoes with the hovered suit highlight

Debug Info:

The highlighting uses these CSS classes:

  • .highlight-primary - Yellow/orange highlight
  • .highlight-secondary - Blue highlight
  • .highlight-suit-0 through .highlight-suit-6 - Suit-specific colors
  • .highlight-doubles - Pink/rose for doubles
