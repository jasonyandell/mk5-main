Texas 42 - Manual Domino Highlighting Test

Test 1: Bidding Phase - Color Highlighting

Step 1: Open the app at http://localhost:60101
Step 2: Click on the "Actions" tab at the bottom
Step 3: You should see "Your Hand" section with 7 dominoes
Step 4: Hover your mouse over any domino (e.g., 3-5)
Expected Result:
- All dominoes with that suit (e.g., 3s) should highlight with the suit color:
  • Blanks (0) = Gray
  • Ones (1) = Red
  • Twos (2) = Orange
  • Threes (3) = Yellow
  • Fours (4) = Green
  • Fives (5) = Blue
  • Sixes (6) = Purple
  • Doubles = Pink
- A colored badge should appear in the top-right showing the suit name

Test 2: Trump Selection - Same Color Highlighting

Step 1: Click "30" to make a bid
Step 2: You should now see "Select Trump" with trump buttons
Step 3: Hover over any trump button (e.g., "Threes")
Expected Result:
- The trump button should show a pointing finger animation
- All dominoes with that suit should highlight with the SAME colored highlighting as during bidding
- The dominoes should have the colored background (not just a border)

Debugging Tips

Open the browser DevTools (F12) and check:
  • Console for any JavaScript errors
  • Elements tab - hover over a domino and check if it has classes like highlight-suit-3 or highlight-primary
  • The domino element should have both a highlight prop (primary/secondary) AND a highlightColor prop (suit-0 through suit-6)

Current Status

We've updated the code so that both bidding and trump selection phases should show the same colored suit highlighting. The getHighlightColor function now returns suit colors for both phases.
